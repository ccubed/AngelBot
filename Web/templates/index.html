AngelBot
 Features
AngelBot has one goal. External Services. It does that really well. Want to interface with Anilist on Discord? AngelBot is your bot.
Want to use it with other bots? That's okay too. We can dig it. The command prefix for AngelBot can be anything. AngelBot is robo friendly.
AngelBot has per guild settings. Additionally, each command set can be locked to specific channels in your guild if you like.
AngelBot has a flexible configuration system. You can control channel locks through AngelBot or by simply restricting what channels AngelBot can hear.
